# 2016 Internet Time Machine

A fake 2016 internet inside a fake Chrome browser. Search "ChatGPT" and watch Google have a full crisis about it. Google, YouTube, Twitter — pixel-accurate, stuck in 2016, deeply confused by anything you actually care about.

## Dev

```bash
npm run dev      # start dev server
npm run build    # production build
npm run lint     # lint
```

Requires `VITE_YOUTUBE_API_KEY` in `.env` for live YouTube search (optional, falls back gracefully).
